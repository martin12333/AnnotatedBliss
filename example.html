AnnotatedBliss V2 Docs

Example

Source:

Hello and <bliss symbols="feeling arrowintobuilding">welcome</bliss> to the home of <bliss symbols="language worldwriting">Blissymbols</bliss>.

Displays as:

Hello and welcome to the home of Blissymbols.

How to install

  1. First upload the CSS (annotatedBlissV2.css) and Javascript (annotatedBlissV2.js) files to your server.
  2. Next modify the 'symbolPath' and 'symbolFormat' variables in the Javascript file to the correct values for your site (more info in the Javascript file).
  3. Next include the CSS file in your <head> section with the following code: <link rel="stylesheet" type="text/css" href="/your/path/to/annotatedBlissV2.css"></link>.
  4. Then include the Javascript file in the <head> section with the following code: <script type="text/javascript" src="your/path/to/annotatedBlissV2.js"></script>

How to use

Wherever you wish some annotated Bliss to appear simply use the tags <bliss symbols="symbol1 symbol2 symbol3">Annotation goes here</bliss>. The symbols should be listed as their filename of the image without their extension (.png, .gif, .jpg etc).
